# Cartella musica

Inserisci qui il file audio rinominandolo **`ambient.mp3`**.

## Requisiti
- Formato: MP3 (128–192 kbps, mono o stereo)
- Durata consigliata: ≥ 3 minuti (il file viene messo in loop automatico)
- Licenza: assicurati di avere il diritto di utilizzo

## Suggerimenti per brani royalty-free
- **Freesound.org** – cerca "ambient relaxing loop"
- **Free Music Archive** – filtro CC0 / Public Domain
- **Pixabay Music** – sezione "Meditation & Ambient"
- **Incompetech (Kevin MacLeod)** – licenza CC BY

Una volta aggiunto il file, il player apparirà come pulsante flottante
in basso a destra su tutte le pagine (desktop e mobile).
