# Plan 03-01: Componente WpPageShell

## Scope
Creare un componente React condiviso che incapsula il boilerplate Elementor/WordPress comune a tutte le pagine. Refactoring della pagina `biografia` come prova concetto. Questo permette di aggiungere nuove sezioni con un `page.tsx` di ~10 righe invece di ~1000.
